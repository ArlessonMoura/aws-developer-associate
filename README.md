# ☁️ AWS Developer Associate Manual

> _"A melhor forma de aprender é estudar, construir, errar, corrigir e compartilhar."_

Este projeto nasceu exatamente dessa ideia.

Ao longo da minha preparação para a certificação **AWS Certified Developer – Associate**, percebi que meus resumos estavam espalhados em diversos arquivos, anotações e documentos. Em vez de mantê-los apenas para uso pessoal, decidi transformá-los em algo mais organizado, agradável de estudar e, principalmente, útil para outras pessoas.

Mais do que um repositório de anotações, este manual representa minha forma de aprender: **escrevendo, implementando e compartilhando conhecimento**.

---

## 🚀 O objetivo

Este manual busca reunir, em um único lugar, os principais bizus para se preparar para a certificação AWS Developer - Associate.

Aqui você encontrará conteúdos sobre:

- ☁️ Computação em Nuvem
- ⚡ AWS Lambda
- 🗄️ Amazon DynamoDB
- 🔐 IAM
- 📬 SNS e SQS
- 📡 EventBridge
- 📊 CloudWatch
- 🏗️ CloudFormation e AWS SAM
- 🔒 Segurança
- 🧠 Arquitetura Serverless
- 📖 Resumos para revisão da certificação

O foco não é apenas decorar conceitos para uma prova, mas entender **como as tecnologias funcionam em conjunto para construir aplicações escaláveis, resilientes e modernas.**

---

# ❤️ Compartilhar conhecimento

Sempre acreditei que conhecimento só faz sentido quando é compartilhado.

Durante minha transição para a Engenharia de Software, encontrei inúmeras pessoas que dedicaram seu tempo para escrever artigos, gravar vídeos, responder dúvidas em fóruns e criar conteúdos gratuitos.

Este projeto é uma pequena forma de retribuir tudo isso.

Se este manual ajudar alguém a compreender um serviço da AWS, resolver uma dúvida ou conquistar sua certificação, então ele já terá cumprido seu propósito.

---

# ☁️ Minha paixão pela AWS

A computação em nuvem mudou completamente a forma como construímos software.

O que antes exigia servidores físicos, infraestrutura complexa e altos investimentos, hoje pode ser criado em poucos minutos utilizando serviços gerenciados.

Foi impossível não se apaixonar por esse ecossistema.

Cada serviço estudado revela uma nova forma de resolver problemas reais com simplicidade, escalabilidade e alta disponibilidade.

A AWS não é apenas uma plataforma de infraestrutura.

Ela representa uma maneira diferente de pensar arquitetura de software.

# 🚀 Executando o projeto

Clone o repositório:

```bash
git clone <repo>
```

Instale as dependências:

```bash
npm install
```

Execute o ambiente de desenvolvimento:

```bash
npm run dev
```

Abra no navegador:

```
http://localhost:5173
```
